# vscode

```git
git branch  //查看当前分支
git branch name //创建新分支
git checkout name  //切换新分支
git merge name  //合并新分支
git log
git stash	临时保存工作区修改，便于切换
git status	查看当前工作区和暂存区状态
git add .  //添加暂存库
git commit -m "你的提交信息"  //提交
git push origin main //推送到远程仓库
git remote add origin SSH格式地址
```
